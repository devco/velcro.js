Routing

Routing is done using an instance of velcro.Router. It listens for changes to the current URL and updates a given element with content from the resulting route.

A router uses 3 components in order to do your bidding. It uses a velcro.State instance to listen for changes in the URL. When it is initialised or changes and a match is found, it then uses a velcro.App instance to bind the model returned from the matched route's controller to the content returned from its velcro.View instance.

Setting Up a Router

To set up a router we use an instance of velcro.Router.

var router = new velcro.Router({
    app: new velcro.App,
    state: new velcro.State,
    view: new velcro.View
});
