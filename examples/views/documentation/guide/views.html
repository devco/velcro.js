Views Functional UI separation.

Views allow you to separate your application into smaller chunks of display logic. A single view instance is set up to render multiple views using the same conventions. The router uses a view instance to render views into its bound container. The include and extend bindings use view instances to transform the elements.

To setup a view instance, simply create a new velcro.View:

var view = new velcro.View(options);
