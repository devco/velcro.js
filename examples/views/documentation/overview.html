velcro.js Your friendly Javascript framework nextdoor.

Velcro gives you a complete MVC solution for building RESTful JavaScript web applications. Although there aren't any dependencies, an AMD library like RequireJS is recommended.

Features include:

  • Full MVC separation.
  • Full AMD / CommonJS support.
  • Classical inheritance support using a similar method to John Resig's.
  • Complete Model / Collection and relationship management.
  • View component allowing views to be loaded from multiple sources including their own files and DOM elements.
  • Attribute bindings akin to AngularJs and KnockoutJS.
  • HTTP REST Client with built-in JSON support.
  • Simple and flexible routing with full support for both browser state and hashchange events.
  • Convention over configuration, yet still allowing you to configure what you want, when you want.

Bower Installation

You can install directly from Bower:

bower install velcro.js

Or add it to your Bower component.js file:

{
  "dependencies": {
    "velcro.js": "latest"
  }
}

Manual Installation

The compiled Velcro source files are located in the dist/ folder. How you consume them is up to you.

Using with RequireJS

Use with RequireJS just as you would any other AMD module:

require(['path/to/velcro'], function(velcro) {
  ...
});

The beauty of using RequireJS - or your choice of AMD library - really shines when you begin to separate out the different parts of your application, especially models. It also allows you to very easily optimise your whole application. See the Optimisation section in the guide for more information.
